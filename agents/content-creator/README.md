# Content Creator Agent

Executes content based on strategist briefs.

## What This Agent Does

- **Social posts** - Write platform-specific content
- **Newsletter drafts** - Create email content
- **Script writing** - Draft video/podcast scripts
- **Caption writing** - Engaging captions for visual content

## Key Integrations

Works with:
- **Content Strategist** - Receives briefs and direction
- **Audience Intel** - Queries for voice-of-customer language
