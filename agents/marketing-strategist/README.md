# Marketing Strategist Agent

High-level marketing decisions and campaign planning.

## What This Agent Does

- **Campaign planning** - Design marketing campaigns
- **Positioning** - Refine messaging and positioning
- **Funnel optimization** - Improve conversion paths
- **Launch strategy** - Plan product/offer launches

## Key Integrations

Works with:
- **Audience Intel** - Research for strategic decisions
- **Content Strategist** - Aligns content with campaigns
