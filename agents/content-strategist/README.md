# Content Strategist Agent

Plans your content calendar and strategy.

## What This Agent Does

- **Content calendar** - Plan weekly/monthly content themes
- **Content repurposing** - Turn one piece into many formats
- **Platform strategy** - Optimize content for each platform
- **Performance analysis** - Review what's working

## Key Integrations

Works with:
- **Audience Intel** - Query for trending topics and pain points
- **Content Creator** - Hands off briefs for execution
