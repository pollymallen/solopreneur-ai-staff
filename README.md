# Solopreneur AI Staff - Starter Kit

Build your AI staff with Claude Code. This repo contains everything you need to create a team of AI agents for your business.

## Quick Start

1. **Clone this repo**
   ```bash
   git clone https://github.com/pollymallen/solopreneur-ai-staff.git
   cd solopreneur-ai-staff
   ```

2. **Open in Claude Code**
   ```bash
   claude
   ```

3. **Follow the course** at [pollymallen.github.io/claude-code-for-solopreneurs](https://pollymallen.github.io/claude-code-for-solopreneurs) to build each agent

## What's Inside

```
├── CLAUDE.md              # Your AI staff's shared memory
├── agents/                # Agent-specific instructions

│   ├── content-strategist/
│   ├── content-creator/
│   └── marketing-strategist/
├── .claude/commands/      # Custom slash commands
├── templates/             # Reusable templates for your agents
└── exercises/             # Practice files for the course
```

## Prerequisites

- [Claude Pro or Max subscription](https://claude.ai) ($20/month)
- [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code) installed
- Basic terminal comfort
